# Contributing to DC Tools

Thank you for your interest in DC Tools! 

## How to Contribute

Since DC Tools is proprietary software, we don't accept code contributions. However, you can still help:

### 🐛 Report Bugs
Found a bug? Let us know at [bugs.deniscasian.com](https://bugs.deniscasian.com)

### 💡 Request Features
Have an idea for a new tool? Submit it at [qa.deniscasian.com](https://qa.deniscasian.com)

### 📢 Spread the Word
- Share DC Tools with friends and colleagues
- Write about us on your blog
- Give us a shoutout on social media

### ⭐ Star the Repository
If you find DC Tools useful, please star this repository!
